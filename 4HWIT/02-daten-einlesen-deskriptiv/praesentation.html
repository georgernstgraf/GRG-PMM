Die Pipe |>

Nimmt das Ergebnis links und gibt es als erstes Argument rechts ein.

# Ohne Pipe — verschachtelt, schwer zu lesen
summarise(filter(iris, Sepal.Length > 5), m = mean(Sepal.Length))

# Mit Pipe — linear, natürlich zu lesen
iris |>
  filter(Sepal.Length > 5) |>
  summarise(m = mean(Sepal.Length))

NA — fehlende Messwerte

NA = Not Available. Nicht Null, nicht leer — ein eigener Zustand.

x <- c(1, 2, NA, 4, 5)
mean(x)               # → NA!  (ansteckend)
mean(x, na.rm = TRUE) # → 3   (NA entfernt)
is.na(x)              # → FALSE FALSE TRUE FALSE FALSE
Wichtig

NA steckt jede Rechnung an. na.rm = TRUE entfernt es — aber frage zuerst: warum fehlen Werte?

Deskriptive Statistik

Beschreiben, was man hat — nicht schließen.

Zwei Fragen stehen im Zentrum:

  1. Lage: Wo liegen die Werte? (Mittelwert, Median)
  2. Streuung: Wie weit streuen sie? (sd, IQR)

Beides braucht man — eine Zahl allein reicht nie.

Lagemaße — wo liegt die Verteilung?

MaßDefinitionIntuition

median(iris$Sepal.Length)   # → 5.8

Eselsbrücke

Der Mittelwert wird in Richtung des Schwanzes gezogen. Schwanz rechts → Mittel > Median → rechtssteil.

Bei schiefen Verteilungen / Ausreißern: Median.
Bei symmetrischen Verteilungen: Mittelwert.

Warum Streuung genauso wichtig ist

gruppe_a <- c(48, 49, 50, 51, 52)    # eng
gruppe_b <- c(0, 25, 50, 75, 100)    # weit

mean(gruppe_a)   # → 50
mean(gruppe_b)   # → 50

Beide haben Mittelwert 50! Völlig verschiedene Verteilungen.

→ Eine Zahl allein beschreibt eine Verteilung nie.

Standardabweichung — die Intuition

Warum quadriert die Varianz?

Einfache Abweichungen heben sich auf (positiv vs. negativ = 0). Quadrieren macht alles positiv. Aber: cm² statt cm. Wurzel ziehen → Standardabweichung zurück in Original-Einheiten.

68-95-Regel (bei Normalverteilung)

~68% innerhalb ±1 sd · ~95% innerhalb ±2 sd · ~99,7% innerhalb ±3 sd

Darauf basieren später die Regelkarten-Grenzen (KM 8).

group_by() + summarise()

Pro Gruppe eine Kennzahl — wie SQL GROUP BY.

iris |>
  group_by(Species) |>
  summarise(
    anzahl    = n(),
    mittel_SL = mean(Sepal.Length),
    sd_SL     = sd(Sepal.Length)
