R & RStudio

R = die Programmiersprache für Statistik (seit 1993).

RStudio = die IDE (4 Fenster: Konsole, Skript, Environment, Plots).

Installation

1. R von cran.r-project.org
2. RStudio von posit.co/download
Wichtig: erst R, dann RStudio.

Die Konsole

R ist ein Taschenrechner mit Gedächtnis:

2 + 3 * 4    # → 14
2^10         # → 1024
Prompt > unten links in RStudio

Befehl eintippen, Enter drücken, Ergebnis erscheint.

Das Wichtigste: Alles ist ein Vektor

In R ist der Vektor das Grundelement — nicht der Skalar.

Du baust einen Vektor mit c() (combine).

c(1, 2, 3)      # → 1 2 3
c(1, 2, 3) * 2  # → 2 4 6

Keine Schleife. Jedes Element wird einzeln multipliziert — das ist Vektorisierung.

Vektor trifft Vektor

c(1, 2, 3) + c(10, 100, 1000)  # → 11 102 1003
Recycling

Ungleiche Länge? R verlängert den kürzeren Vektor automatisch:

c(1, 2, 3, 4) + c(10, 100)  # → 11 102 13 104

Aber: Längen müssen sauber aufgehen, sonst gibt es eine Warnung.

Noch mehr Vektoren

1:6                         # → 1 2 3 4 5 6
seq(0, 10, by = 2)          # → 0 2 4 6 8 10
rep(c(1,2), times = 3)      # → 1 2 1 2 1 2
rep(c(1,2), each  = 3)      # → 1 1 1 2 2 2

1:3 = aufeinanderfolgend · seq() = mit Schrittweite · rep() = wiederholen · c() = kombinieren

Werte speichern — mit dem Pfeil

messwerte <- c(8.1, 7.9, 8.0, 8.2, 7.8)
messwerte                # → 8.1 7.9 8.0 8.2 7.8
mean(messwerte)          # → 8  (Mittelwert)
Shortcut in RStudio: Alt + -

Der Zuweisungsoperator ist <- (nicht =, nicht ==).

= nur in Funktionsargumenten: seq(0, 10, by = 2)

Die Regel: gleichartig vs. gemischt

vector und matrix = ein Typ für alles.
data.frame und list = Spalten/Elemente dürfen verschiedene Typen haben.

Coercion — was passiert, wenn man Typen mischt?

Die Hierarchie: logical → integer → double → character

c(1, "a")      # "1" "a"  — character gewinnt
c(TRUE, 1, "a") # "TRUE" "1" "a"  — alles Text

Der höchste Typ in der Kette gewinnt — und zwingt alle anderen mit.

list — Fächer & Boxen

Eine Liste ist ein Regal mit Fächern. In jedem Fach liegt eine Box, in der Box ein Wert.

modell <- list(
  koeffizient = c(7.8, 0.3),
  p_wert      = 0.012
)
SchreibweiseWas du bekommst
modell[1]die Box (Liste) — Inhalt noch verpackt
modell[[1]]den Inhalt (den Vektor selbst)
modell$koeffizientden Inhalt per Namen (Kurzform von [[ )

Beweis: identical(modell[1][[1]], modell[[1]]) → TRUE
Erst die Box, dann öffnen = direktes Öffnen.

factor — Kategorien für Statistik

Ein Factor sieht aus wie Text — drinnen ist er Integer mit Labels.

f <- factor(c("gut", "schlecht", "gut"))
f               # → gut schlecht gut
as.integer(f)   # → 1 3 1  (die Codes!)

Intern: 1 = "gut", 2 = "mittel", 3 = "schlecht" (alphabetisch!).

Modelle wie aov() und lm() brauchen Factor für diskrete Gruppen.

Die Factor-Falle

Aufgepasst

Zahlen, die als Text in einem Factor stecken, sind nach der Rückumwandlung nicht die Zahlen!

as.numeric(factor(c("10", "20", "30")))          # → 1 2 3  (CODES, nicht Werte!)
as.numeric(as.character(factor(c("10","20"))))    # → 10 20   (so ist es richtig)

Merke: as.numeric(f) = Codes · as.numeric(as.character(f)) = Werte (nur wenn Labels Zahlen sind).
