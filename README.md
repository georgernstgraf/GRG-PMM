# GRG-PMN

**Projektmanagement und Netzwerktechnik**  
HTL Spengergasse — Abteilung Wirtschaftsingenieure (Technisches Management & Umwelt)

---

## Übersicht

Dieses Repository enthält Unterrichtsmaterialien für das Fach **Projektmanagement und Netzwerktechnik (PMN)** im Schuljahr 2025/26. Es dient als zentrale Anlaufstelle für Unterrichtsinhalte, Übungen und Projekte.

## Aufbau

| Verzeichnis | Inhalt |
|-------------|--------|
| `4AHWIT/` | Unterrichtsordner 4. Klasse — datierte Einträge (`YYYY-MM-DD_thema`) mit Code, Übungen und Notizen |
| `5AHWIT/` | Unterrichtsordner 5. Klasse |
| `Unterlagen/` | Thematische Referenzmaterialien und Skripten |
| `docs/` | Zentrale Dokumente (Lehrplan, Metadaten, Projektangaben) |

### Rechtliche Grundlagen

| Datei | Beschreibung |
|-------|-------------|
| `docs/lehrplan/Lehrplan_TechnischesManagement.pdf` | Offizielles Schul-PDF (HTL Spengergasse) |
| `docs/lehrplan/2015-09-17_BGBl-II-262_Anlage-1.28_HTL-Wirtschaftsingenieure-TechnischesManagement.pdf` | Signiertes RIS-Original |
| `docs/lehrplan/2015-09-17_BGBl-II-262_Anlage-1_Allgemeiner-Teil-HTL.pdf` | Allgemeiner Teil aller HTL-Lehrpläne |
| `docs/lehrplan/METADATA.md` | Metadaten und RIS-Verweise zum Lehrplan |

Der Lehrplan basiert auf **BGBl. II Nr. 262/2015** (Lehrplanpaket HTL 2015), Anlage 1.28 — Höhere Lehranstalt für Wirtschaftsingenieure – Technisches Management.

> **Hinweis:** Dieses Repository ist öffentlich. Es enthält **keine** Test-Angaben.
