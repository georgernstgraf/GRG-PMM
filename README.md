## Übersicht

PMM ist das Maturafach der WIT-Abteilung. Es verbindet **statistische Datenanalyse mit R** und **betriebliches Umwelt- und Facility Management** zur ganzheitlichen Prozessoptimierung.

| Themenfeld | Inhalte |
|------------|---------|
| Statistische Methoden | Hypothesentests, SPC, Prozessfähigkeit, Versuchsmethodik (DOE), Zuverlässigkeit |
| Umweltmanagement | ISO 14001, Ökobilanz (LCA), Stoffflussanalyse, Umweltrecht |
| Facility Management | DIN EN 15221, Instandhaltung, Energiemanagement (ISO 50001) |
| Werkzeug | R / RStudio — von deskriptiver Statistik bis zur industriellen Versuchsplanung |

Der Lehrplan folgt **BGBl. II Nr. 262/2015, Anlage 1.28, Abschnitt 5** — rechtliche Grundlage für den gesamten Unterricht.
