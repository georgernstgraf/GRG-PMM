Zufallsvariablen & Wahrscheinlichkeit

Das Vokabular, ohne das keine Verteilung, kein Konfidenzintervall und kein t-Test funktioniert: Was ist eine Zufallsvariable, was unterscheidet diskret von stetig, und warum ist eine Dichte keine Wahrscheinlichkeit? Wer das sicher hat, liest jede Statistik-Formel.

Warum diese Lektion

Die gesamte Statistik — vom Einstiegstest bis zur Zuverlässigkeitsanalyse im JG5 — baut auf einem einzigen Begriff auf: der Zufallsvariable. In Track 1 lernst du die Verteilungen (Binomial, Poisson, Normal, Exponential) kennen. Bevor eine davon drankommt, brauchst du das Gerüst: diskret vs. stetig, Wahrscheinlichkeitsfunktion vs. Dichtefunktion, und die Verteilungsfunktion F(x) als gemeinsame Sprache. Das ist diese Lektion.

Die eine Idee

Eine Zufallsvariable ordnet jedem Ausgang eines Zufallsexperiments eine Zahl zu. Der Trick der Statistik: Wir beschreiben nicht mehr jeden Ausgang einzeln, sondern die Verteilung der Zahlen — über zwei Funktionen, f(x) und F(x). Wer den Unterschied dieser beiden Funktionen verstanden hat, hat 80 % des Formal-Zeugs gemeistert, das im Einstiegstest abgefragt wird.

Was ist eine Zufallsvariable?

Ein Zufallsexperiment (Würfeln, Bauteil auf Lebensdauer prüfen, Messen eines Durchmessers) hat Ausgänge. Eine Zufallsvariable X ist eine Vorschrift, die jedem Ausgang eine Zahl zuordnet:

ExperimentAusgangZufallsvariable X
Würfel werfen1, 2, 3, 4, 5, 6gewürfelte Augenzahl
Bauteil prüfenfunktioniert / defekt1 = defekt, 0 = i.O.
Durchmesser messenjeder reelle Wert um den Sollwertgemessener Wert (mm)
Bauteil betreibenZeit bis AusfallLebensdauer (h)

Der Gewinn: Statt über Ausgänge reden wir über Zahlen und können rechnen — Mittelwert, Streuung, Wahrscheinlichkeiten für Wertebereiche.

Diskret vs. stetig — die entscheidende Gabelung

Zufallsvariablen kommen in zwei Geschmacksrichtungen, und fast jede Formel ändert sich je nachdem:

diskretstetig
Werteabzählbar (1, 2, 3, …)überabzählbar (jeder reelle Wert in einem Intervall)
BeispieleWürfel, „Anzahl defekter Stücke"Durchmesser, Lebensdauer, Reaktionszeit
P(X = x)kann > 0 seinstets 0 für jeden einzelnen Wert
Funktion f(x)Wahrscheinlichkeitsfunktion: f(x) = P(X = x)Dichtefunktion: f(x) ist keine Wahrscheinlichkeit
Der Schock für Anfänger

Bei einer stetigen Zufallsvariable ist die Wahrscheinlichkeit, genau einen Wert zu treffen (z. B. Durchmesser = 10,000000 mm), null. Das klingt paradox — natürlich ist der gemessene Wert irgendeine Zahl. Aber es gibt unendlich viele mögliche Zahlen, und jede einzelne trägt davon null Gewicht. Wahrscheinlichkeiten entstehen erst über Intervalle — als Fläche unter der Dichtefunktion.

Im Bild: Punkte vs. Fläche

Links: bei einer diskreten Verteilung sitzen Wahrscheinlichkeiten als Punkte (Balken) auf den einzelnen Werten. Rechts: bei einer stetigen Verteilung ist die Wahrscheinlichkeit die Fläche unter der Kurve zwischen zwei Werten — ein einzelner Punkt hat keine Fläche, also P(X = x) = 0.

Verständnis
Eine stetige Zufallsvariable: was gilt für P(X = x)?
  • P(X=x) ist stets null
  • P(X=x) ist stets eins
  • P(X=x) ist stets klein
  • P(X=x) ist stets gross
Richtig. Bei stetigen ZV hat ein einzelner Wert keine Fläche — die Wahrscheinlichkeit steckt in Intervallen, nicht in Punkten.
Falsch. Bei stetigen ZV ist P(X = x) = 0 für jeden einzelnen Wert; Wahrscheinlichkeiten entstehen erst über Intervalle (Fläche unter der Dichte).
Gegenprobe
Eine diskrete Zufallsvariable …
  • nimmt abzählbar Werte an
  • nimmt stetig Werte an
  • nimmt keine Werte an
  • nimmt alle Werte an
Richtig. „Abzählbar" heißt: die Werte lassen sich aufzählen (1, 2, 3, …). Beim Würfel sind es sechs, bei „Anzahl defekter Stücke" jede nichtnegative ganze Zahl.
Falsch. Diskret = abzählbar viele Werte (z. B. 1–6 beim Würfel). Stetig = überabzählbar, jeder Wert in einem Intervall.

Die zwei Funktionen: f(x) und F(x)

Jede Verteilung wird durch zwei Funktionen beschrieben. Sie werden im Einstiegstest (Frage 8) direkt abgefragt — und in jeder Folgelektion vorausgesetzt.

f(x) — Wahrscheinlichkeits- bzw. Dichtefunktion

Diskret: f(x) = P(X = x). Das ist direkt eine Wahrscheinlichkeit — z. B. f(3) = 1/6 beim Würfel.

Stetig: f(x) ist die Dichtefunktion. Sie ist keine Wahrscheinlichkeit und darf sogar größer als 1 sein. Erst das Integral (die Fläche) über ein Intervall liefert eine Wahrscheinlichkeit:

P(a ≤ X ≤ b) = ∫ f(x) dx  (von a bis b)

F(x) — die Verteilungsfunktion

Die Verteilungsfunktion gilt für diskret UND stetig und ist der große Vereiniger:

F(x) = P(X ≤ x)

Bei einer stetigen ZV ist F(x) das Integral über die Dichte von −∞ bis x. Und die Dichte ist die Ableitung von F(x):

F(x) = ∫ f(t) dt  ⇒  f(x) = F'(x)

Eselsbrücke — Kleiner-als

Das F steht für die kumulierte Wahrscheinlichkeit „bis kleiner-gleich x": F(x) = P(X ≤ x). Die Funktion steigt monoton von 0 (ganz links) nach 1 (ganz rechts) — mehr geht nicht, weil Wahrscheinlichkeiten maximal 1 sind.

Didaktik — typischer Schülerirrtum

Schüler verwechseln f(x) und F(x) regelmäßig. Die Frage „P(X ≤ 5)" verlangt F(5), nicht f(5). Trenne schon hier scharf: f = Dichte/Wahrscheinlichkeit an einem Punkt, F = alles bis zu diesem Punkt. Im Einstiegstest (Frage 8) ist genau diese Unterscheidung Teil der korrekten Antwort.

Verständnis
Die Verteilungsfunktion F(x) gibt an …
  • P(X ≤ x) direkt an
  • P(X = x) direkt an
  • P(X > x) direkt an
  • P(X ≠ x) direkt an
Richtig. F(x) = P(X ≤ x) — die kumulierte Wahrscheinlichkeit bis zum Wert x. Funktioniert für diskrete wie stetige ZV.
Falsch. F(x) = P(X ≤ x). Für P(X = x) brauchst du f(x) (diskret) bzw. es ist 0 (stetig).
Falle
Die Dichtefunktion f(x) einer stetigen ZV ist …
  • keine Wahrscheinlichkeit direkt
  • direkt eine Wahrscheinlichkeit
  • stets eine Konstante null
  • immer unendlich gross
Richtig. f(x) selbst ist keine Wahrscheinlichkeit (darf sogar > 1 sein). Erst die Fläche unter f über einem Intervall ist P(a ≤ X ≤ b).
Falsch. Die Dichte f(x) ist kein Wahrscheinlichkeits­wert. Die Wahrscheinlichkeit steckt in der Fläche unter der Kurve, nicht im Kurvenwert selbst.

R-Praxis — eine Zufallsvariable simulieren

Bevor wir in den nächsten Lektionen die konkreten Verteilungen mit R berechnen, simulieren wir hier eine diskrete ZV direkt: den Würfel. Die Funktion sample() zieht zufällig aus einer Menge — unser erstes echtes Zufallsexperiment in R.

# 100-mal wuerfeln: Ziehen MIT Zuruecklegen aus 1:6
set.seed(42)                       # reproduzierbar — gleiche Zufallszahlen
wurf <- sample(1:6, size = 100, replace = TRUE)
head(wurf, 10)                     # die ersten 10 Ergebnisse
R-Ergebnis zeigen (erst selbst vorhersagen) 
 [1] 1 5 2 5 6 4 3 3 6 4

Jetzt zählen wir, wie oft jede Augenzahl kam — das ist die empirische Häufigkeitsverteilung, ein Schatten der theoretischen Wahrscheinlichkeitsfunktion f(x):

table(wurf)                       # absolute Haeufigkeit je Wert
table(wurf) / 100                 # relative Haeufigkeit ≈ P(X = x)
Output 
wurf
 1  2  3  4  5  6
16 18 18 22 13 13

        1    2    3    4    5    6
     0.16 0.18 0.18 0.22 0.13 0.13

Die relativen Häufigkeiten pendeln um 1/6 ≈ 0,167 — das ist das Gesetz der großen Zahlen im Kleinen (kommt ausführlich in L0008). Mit 10 000 Würfen würde es noch viel enger um 0,167 liegen.

Vorschau: das d/p/q/r-Schema

Für jede benannte Verteilung (Binomial, Poisson, Normal, …) hält R vier Funktionen bereit, die alle gleich aufgebaut sind — mit einem Buchstaben-Prefix:

PrefixliefertBeispiel (Normal)
dDichte/Wahrscheinlichkeit f(x)dnorm(0) → f(0)
pVerteilungsfunktion F(x) = P(X ≤ x)pnorm(0) → 0,5
qQuantil — Umkehrung von Fqnorm(0,975) → 1,96
rZufallszahlen (simulieren)rnorm(5)

Merkregel: d = density, p = probability (cumul.), q = quantile, r = random. Ab L0004 wenden wir das auf konkrete Verteilungen an; sample() oben war der „handgemachte" Vorläufer von r.

Didaktik — warum sample() vor den Prefixen

Im Unterricht lohnt es sich, den Würfel erst mit sample() zu simulieren, bevor rbinom() kommt. So sehen Schüler, dass eine Verteilung ein Zufallsexperiment ist, das sie selbst bauen können — und die d/p/q/r-Prefixe sind dann nur noch Komfortfunktionen für dasselbe Prinzip.

R-Praxis
Was liefert table(sample(1:6, 100, replace = TRUE))?
  • Haeufigkeit je Augenzahl
  • Summe aller Augenzahlen
  • Mittelwert der Augenzahl
  • Maximum der Augenzahlen
Richtig. sample(1:6, 100, replace = TRUE) würfelt 100-mal, table() zählt je Wert — die empirische Häufigkeitsverteilung.
Falsch. table() zählt die Häufigkeit je Wert. Für die Summe bräuchtest du sum(), für den Mittelwert mean().

Hausübung (für dich — Selbststudium)

  1. Simuliere sample(1:6, 1000, replace = TRUE) und vergleiche die relativen Häufigkeiten mit 1/6. Werden sie näher an 0,167 als bei 100 Würfen? (Gesetz der großen Zahlen —Vorgriff auf L0008.)
  2. Erkläre mit eigenen Worten, warum bei einer stetigen Zufallsvariable P(X = x) = 0 gilt, obwohl der gemessene Wert eine konkrete Zahl ist.
  3. Welche der beiden Funktionen — f(x) oder F(x) — würdest du verwenden, um „P(X ≤ 5)" zu berechnen? Begründe.
  4. Teste in R: pnorm(0) und dnorm(0). Welcher Wert ist eine Wahrscheinlichkeit, welcher nicht? (Die Normalverteilung kommt in L0005 — hier nur „schnuppern".)
  5. Vorhersage: Ist dnorm(0) größer oder kleiner als 1? Prüfe und erkläre mit „Dichte ist keine Wahrscheinlichkeit".
