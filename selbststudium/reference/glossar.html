Glossar — R & industrielle Statistik

Die Begriffe, an die sich alle Lektionen und Learning Records halten. Ein Begriff wird erst aufgenommen, wenn er verstanden wurde — nicht beim ersten Vorstellen. Dieses Glossar wächst mit.

Konventionen

  • Sprache: Deutsch für Erklärtexte; R-/Statistik-Fachbegriffe teils englisch (data frame, p-value, ANOVA).
  • „Mittelwert“ meint hier immer das arithmetische Mittel, sofern nicht anders genannt.
  • „Stichprobe“ = Teilmenge einer Grundgesamtheit, gezogen zur Untersuchung.
  • p-Wert ≠ Wahrscheinlichkeit: ein p-Wert ist keine Wahrscheinlichkeit, dass die Nullhypothese stimmt. (Wird bei Einführung ergänzt.)

Statistik — Grundbegriffe

Mittelwert (arithmetisches Mittel)
Summe aller Werte geteilt durch ihre Anzahl. Empfindlich gegenüber Ausreißern.
Vermeide: „Durchschnitt", „der Mean".
Median
Wert, der eine der Größe nach sortierte Reihe genau halbiert. Robust gegenüber Ausreißern. Bei gerader Anzahl: Mittel der beiden mittleren Werte.

Wahrscheinlichkeit & Verteilungen (Track 1)

Zufallsvariable (X)
Ein Größe, die jedem Ausgang eines Zufallsexperiments eine Zahl zuordnet. Wir reden danach über die Zahlen statt über Ausgänge — und können rechnen.
Vermeide: „Zufallsgröße" wenn Verwirrung droht (im PMM-Kontext gilt „Zufallsvariable" als kanonisch).
diskret
Eine Zufallsvariable nimmt abzählbar viele Werte an (Würfel, „Anzahl defekter Stücke"). P(X = x) kann für einzelne Werte > 0 sein.
stetig
Eine Zufallsvariable nimmt überabzählbar viele Werte in einem Intervall an (Durchmesser, Lebensdauer). P(X = x) = 0; Wahrscheinlichkeiten entstehen nur über Intervalle (Fläche unter der Dichte).
Wahrscheinlichkeitsfunktion f(x) — nur diskret
Ordnet jedem Wert seine Wahrscheinlichkeit zu: f(x) = P(X = x). Summe über alle Werte ist 1.
Dichtefunktion f(x) — nur stetig
Ein Kurvenwert, der keine Wahrscheinlichkeit ist (darf > 1 sein). Erst die Fläche unter der Kurve über einem Intervall liefert P(a ≤ X ≤ b). f ist die Ableitung der Verteilungsfunktion F.
Vermeide: „Punktegl.<2;4;2;dung der Dichte" — Dichte ≠ Wkeit.
Verteilungsfunktion F(x)
F(x) = P(X ≤ x) — die kumulierte Wahrscheinlichkeit bis x. Gilt für diskret und stetig; steigt monoton von 0 nach 1. Bei stetig ist F das Integral über die Dichte f; f ist die Ableitung von F.
Vermeide: Verwechslung mit f(x) — f = an einem Punkt, F = alles bis zum Punkt.
d/p/q/r (R-Präfixe)
Für jede benannte Verteilung hält R vier Funktionen bereit: d = Dichte/Wahrscheinlichkeit f(x), p = Verteilungsfunktion F(x), q = Quantil (Umkehrung von F), r = Zufallszahlen (simulieren).

R & Programmierung — Grundlagen

Funktion
Benannter, wiederverwendbarer Code-Block, der Argumente aufnimmt und einen Wert zurückgibt. In R: name <- function(arg) { … }.
Variable
Ein Name, der an einen Wert gebunden ist. Zuweisung in R mit <-, z. B. x <- 5.
Vektor
Geordnete Folge gleichartiger Werte — das Grundelement in R. Gebaut mit c(). Selbst ein einzelner Wert ist ein Vektor der Länge 1.
Vektorisierung
Eigenschaft von R, Operationen automatisch elementweise auf ganze Vektoren anzuwenden, ohne explizite Schleife (c(1,2,3) * 2 → 2 4 6).
Paket (package / library)
Sammlung von Funktionen und Daten. Einmalig installiert mit install.packages("name"), pro Sitzung geladen mit library(name).
Coercion (Typ-Erzwingung)
Wenn R Werte unterschiedlichen Typs in c() kombiniert, werden alle auf den höchsten Typ der Hierarchie logical < integer < double < character angepasst. Ein "a" zwingt alles zu character.
matrix
2D-Struktur, homogen — alle Elemente ein Typ. Füllt standardmäßig spaltenweise (byrow = FALSE).
data.frame
2D-Tabelle: Zeilen = Beobachtungen, Spalten = Variablen. Jede Spalte ist ein eigener Vektor mit eigenem Typ — heterogen. Der tägliche Arbeitsplatz in R.
list
1D-Container, beliebige Typen pro Element (auch weitere Strukturen). Zugriff: [ = Box behalten (Liste bleibt Liste), [[ = Box öffnen (Inhalt heraus). Box-Metapher: l[1] = „die Box aus Fach 1", l[[1]] = „öffne die Box, gib den Inhalt".
factor
Kategoriale Variable — äußerlich Text, intern Integer-Vektor mit Labels („levels"). Levels werden alphabetisch geordnet. Braucht ANOVA, χ²-Test, Regelkarten für Gruppierungen.
