## Zeitmodell

- 18 Schulwochen/Semester offiziell; real ~15 Doppelstunden (1 DS/Woche)
  nach Abzug von Feiertagen, Admin, Prüfungen
- **13 echte Unterrichtseinheiten (UE) + 2 reservierte DS**
  (Einstiegstest / Leistungsfeststellung / Admin)
- Ausfälle durch Krankheit/Feiertage sind einkalkuliert; bei Glücksfall
  optionale Bonus-UE
